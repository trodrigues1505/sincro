🌀 Sincronário das 13 Luas

Política de Privacidade

Última atualização:

1. Quem somos

O Sincronário das 13 Luas é um aplicativo web (PWA) desenvolvido e operado por Thiago Rodrigues, disponível em trodrigues1505.github.io/sincro. Este aplicativo é dedicado ao estudo e prática do Calendário da Lei do Tempo — o Sincronário de 13 Luas de 28 dias.

Para dúvidas, solicitações ou exercício de direitos previstos na LGPD, entre em contato pelo e-mail: adm.financ.espacoautonomia@gmail.com

2. Dados coletados

Ao utilizar o aplicativo, coletamos os seguintes dados pessoais:

Dados fornecidos pelo login com Google:

  • Nome completo
  • Endereço de e-mail
  • Foto de perfil (URL pública)
  • Identificador único do Google (UID)

Dados fornecidos voluntariamente pelo usuário:

  • Data de nascimento (para cálculo do Kin Natal)
  • Registros do Cronógrafo Diário (afirmações, intenções, sincronicidades) — armazenados apenas no dispositivo do usuário (localStorage)

Dados gerados automaticamente pelo uso:

  • Data e hora do primeiro acesso e dos últimos acessos
  • Atividades realizadas no app (prece, meditação, e-book, etc.) para fins de gamificação
  • Pontuação, sequência (streak) e título galáctico no sistema de ranking
  • Kins favoritados e histórico de consultas (armazenados apenas no dispositivo)
O Cronógrafo Diário, histórico de consultas e favoritos são armazenados exclusivamente no seu dispositivo (localStorage) e nunca são enviados para nossos servidores.

3. Finalidade do tratamento

Os dados são utilizados exclusivamente para:

  • Identificar e autenticar o usuário no aplicativo
  • Controlar o acesso mediante aprovação do administrador
  • Exibir o perfil do usuário dentro do app (nome e foto)
  • Calcular e exibir o Kin Natal a partir da data de nascimento
  • Operar o sistema de gamificação e ranking público entre usuários
  • Enviar notificações push sobre o kin do dia (somente se o usuário ativar)
  • Permitir que o administrador gerencie acessos e envie comunicados

Não utilizamos os dados para publicidade, não os vendemos e não os compartilhamos com terceiros, exceto com os serviços de infraestrutura descritos abaixo.

4. Compartilhamento com terceiros

Para operar o aplicativo, utilizamos os seguintes serviços de infraestrutura:

  • Google Firebase (autenticação e banco de dados Firestore) — Política de Privacidade do Firebase
  • GitHub Pages (hospedagem do aplicativo) — Política de Privacidade do GitHub
  • Google Fonts (tipografia) — dados de acesso às fontes são processados pelo Google
  • YouTube (meditações e self design) — ao assistir vídeos, a política do YouTube se aplica

Esses provedores processam dados conforme suas próprias políticas e possuem certificações de segurança adequadas (SOC 2, ISO 27001).

5. Base legal (LGPD)

O tratamento dos seus dados é realizado com base nas seguintes hipóteses legais previstas na Lei 13.709/2018:

  • Consentimento (Art. 7º, I) — para coleta de dados de perfil e data de nascimento, mediante aceite expresso na tela de login
  • Legítimo interesse (Art. 7º, IX) — para dados de acesso necessários à segurança e operação do serviço
  • Execução de contrato (Art. 7º, V) — para dados necessários à prestação do serviço contratado

6. Retenção dos dados

  • Conta ativa: dados mantidos enquanto a conta estiver ativa
  • Após exclusão da conta: dados removidos do Firestore em até 30 dias
  • Dados de gamificação: excluídos junto com a conta
  • Dados locais (localStorage): permanecem no dispositivo até que o usuário os apague manualmente ou limpe os dados do navegador
  • Logs de acesso: mantidos por até 90 dias para fins de segurança

7. Seus direitos (LGPD, Art. 18)

Você tem os seguintes direitos em relação aos seus dados pessoais:

  • Confirmação e acesso — saber quais dados temos sobre você
  • Correção — corrigir dados incompletos ou desatualizados
  • Anonimização ou eliminação — solicitar a exclusão dos seus dados
  • Portabilidade — receber seus dados em formato estruturado
  • Revogação do consentimento — retirar o consentimento a qualquer momento
  • Oposição — se opor a tratamento realizado com base em legítimo interesse
Para exercer qualquer um desses direitos, envie um e-mail para adm.financ.espacoautonomia@gmail.com com o assunto "LGPD — [seu direito]". Responderemos em até 15 dias úteis.

Você também pode excluir sua conta diretamente no aplicativo, na aba Perfil → Excluir minha conta.

8. Segurança dos dados

Adotamos as seguintes medidas de segurança:

  • Autenticação via OAuth 2.0 do Google (sem senhas armazenadas diretamente)
  • Comunicação exclusivamente via HTTPS
  • Regras de segurança do Firestore (cada usuário acessa apenas seus próprios dados)
  • Aprovação manual pelo administrador antes de qualquer acesso ao app

9. Crianças e adolescentes

O aplicativo é destinado a maiores de 13 anos. Usuários entre 13 e 17 anos precisam da autorização de um responsável legal para utilizar o serviço. Em caso de identificação de uso por menor sem autorização, a conta será suspensa e os dados excluídos.

10. Cookies e armazenamento local

O aplicativo utiliza localStorage do navegador para armazenar preferências, histórico de consultas e cronógrafo diário exclusivamente no seu dispositivo. Não utilizamos cookies de rastreamento ou publicidade.

11. Alterações nesta política

Podemos atualizar esta política periodicamente. Alterações significativas serão comunicadas via banner no aplicativo. A data da última atualização é sempre exibida no topo desta página.
